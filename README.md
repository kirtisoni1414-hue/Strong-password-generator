# Strong-password-generator
A simple Python-based password generator that creates strong and secure passwords using uppercase letters, lowercase letters, numbers, and special characters.
# Project purpose
The purpose of this project is to develop a simple and secure Password Generator using Python. The application helps users create strong, random passwords containing uppercase letters, lowercase letters, numbers, and special characters. Strong passwords improve the security of online accounts by reducing the risk of unauthorized access and password-guessing attacks. This project also helps students understand the use of Python modules, functions, randomization, user input, and basic cybersecurity concepts.
